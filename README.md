# Documentation for flat-siprec-client and flat-mediasrv

Этот каталог содержит архитектурную документацию двух ключевых компонентов:
- **flat-siprec-client** — клиент SIPREC для записи звонков;
- **flat-mediasrv** — медиасервер, который воспроизводит объявления, управляет конференциями и обрабатывает MGCP.

Документация включает:
- системный контекст каждого сервиса;
- внутреннюю архитектуру (component view);
- словарь терминов (glossary);
- файлы PlantUML диаграмм.

Смотри материалы в каталоге `architecture/` и исходники диаграмм в `diagrams/`.
