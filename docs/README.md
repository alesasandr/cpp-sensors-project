# flat-mediasrv

`flat-mediasrv` — это медиасервер, который получает MGCP-команды от Call Agent и выполняет медиазадачи:
проигрывание объявлений, управление конференциями и генерацию NTFY-событий.  
Сервис также публикует метрики через SNMP и использует общий `io_context` для асинхронной обработки MGCP.

## Основные функции
- обработка MGCP (CRCX / MDCX / DLCX / NTFY);
- управление конференциями (CnfHandler);
- проигрывание объявлений (AnnHandler);
- асинхронная MGCP обработка через MgcpExchange;
- метрики и мониторинг через SNMP Agent.
